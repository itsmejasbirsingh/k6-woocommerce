
# K6 Woocommerce

Place woocommerce orders using k6 script.


## Requirements

- K6
- Wordpress
- Woocommrce


## Installation

- Add a few products in woocommerce.
- Add your project's base URL inside setup method of index.js

```bash
  k6 run index.js
```
    
## Versions used

**K6:** v0.47.0 (go1.21.2, darwin/arm64)

**Wordpress:** Version 6.4.1
